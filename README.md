### To load the config file add the following line to ~/.zshrc

```
# Loading zshrc config file
source ~/dotfiles/loader.sh
```

### Install dependencies and config files

```
make setup
```

### Installing dependencies

```
make dependencies
```

### Installing the config files

```
make install
```
